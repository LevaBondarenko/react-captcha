# acms-react-captcha
## Very simple component for react

###**Install**
>$ npm install acms-react-captcha --save

###**Example usage**

```jsx

<ReactCaptcha
	sitekey = 'your_sitekey'
	lang = 'en'
	theme = 'light'
	type = 'image'
	callback = {() => {}}/>

```
